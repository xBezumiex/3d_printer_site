# 3D Print Lab - Сайт для онлайн-заказа 3D-печати

Дипломный проект: Разработка сайта для онлайн-заказа 3D-печати с калькуляцией стоимости и визуализацией 3D-моделей.

## Возможности
- Загрузка 3D-моделей (STL, OBJ, PLY, GLTF, GLB)
- 3D-визуализация с помощью Three.js
- Калькулятор стоимости печати
- Выбор материалов и параметров
- Форма заказа
